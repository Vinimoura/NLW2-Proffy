## 💡 Sobre o Projeto
Criação do Proffy, uma plataforma para conexão professores e alunos.

## 🚀 Tecnologias utilizadas
O projeto foi desenvolvido utilizando as seguintes tecnologias:

- ReactJS
- React Native
- TypeScript
- HTML5 e CSS3
- NodeJS

- Rota para listar aulas
